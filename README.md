A SupplyDrop plugin for your hcf server.

# Commands
- /supplydrop spawn - spawnea en una zona aleatoria del mapa un cofre con items aleatorios en un rango especificado en la config.yml
- /supplydrop edit - edita lo que puede colocarse en el cofr

# Permissions
- supplydrop.command

# Features
- Broadcast cuando se spawnea el supply
- Broadcast cuando se elimina el anterior supply
- Editar el rango aleatorio en el que se puede spawnear el supply en la config.yml
- Editar la cantidad de items aleatorios que se pueden colocar en el supply
- Editar el tiempo en el que se puede spawnear automaticamente el supply
- Autosave de items en rewards.yml

- Si el chunk no esta cargado por un jugador despues de 1000 bloques no se coloca el supplydrop
